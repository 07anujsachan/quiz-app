Quiz-app

Overview-

This is a quiz application built with React, a popular JavaScript library for building user interfaces. The app fetches trivia questions from the Trivia database API and allows users to answer them. It provides a simple and engaging way for users to test their knowledge on various topics.

Features-

. Dynamic Quiz Questions: Questions are fetched from the Trivia database API, providing a diverse set of topics for users.
. Score Tracking: The app keeps track of the user's score as they progress through the quiz.
. User-Friendly Interface: The interface is intuitive, making it easy for users to navigate and answer questions.

Technologies Used- 
. React: The frontend of the application is built using React, allowing for a modular and efficient development process.
. Trivia Database API: The app fetches trivia questions from the Trivia database API to provide a wide range of questions.
. CSS: Styling is done using CSS for a clean and visually appealing layout.
